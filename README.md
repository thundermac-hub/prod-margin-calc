# Margin Calculator

A lightweight React + Vite + Tailwind app for local margin calculations.

## Quick Start (Local)
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Preview Production Build
```bash
npm run preview
```

## Deploy with Coolify (Dockerfile)
Use the Dockerfile in this repo.

Settings:
- Build pack: `Dockerfile`
- Port: `80`
